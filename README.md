# ⚽ Futhub

**¡Organiza tu pasión, salta a la cancha!**

Futhub es la plataforma definitiva diseñada para conectar jugadores y espacios deportivos. Olvídate de los chats interminables para organizar un partido; con Futhub, gestionar tu equipo y encontrar el lugar perfecto es cuestión de segundos.

### 🚀 ¿Qué puedes hacer en Futhub?

* **Arma tu Dream Team:** Crea y gestiona tu equipo, asigna posiciones y mantén la alineación lista.
* **Reserva al instante:** Explora canchas cercanas, verifica disponibilidad en tiempo real y asegura tu reserva.
* **Juega sin límites:** Conecta con otros apasionados del fútbol y organiza encuentros amistosos o competitivos.

## 🛠️ Instalación y Puesta en marcha

Sigue estos pasos para correr el proyecto en tu entorno local:
### Prerrequisitos
* Java 17 o superior.

### Ejecución
Abre la terminal en la carpeta del proyecto, viaja a la carpeta "back" y ejecuta:

#### En Windows: mvnw spring-boot:run

#### En Mac/Linux: ./mvnw spring-boot:run

### Para el frontend
Vuelve a la carpeta del proyecto y ahora viaja a la carpeta "front" y ejecuta: npm run dev

Entra al link y verás la página desplegada en tu navegador
